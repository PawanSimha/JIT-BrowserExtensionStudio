Goofanizer

Download Extension

circle Overview

Goofanizer emulates mobile and tablet viewports with one click. Switch between device presets, rotate orientation, capture screenshots, and batch export across all presets. Use it for testing responsive layouts, verifying mobile UIs, and capturing cross-device screenshots for documentation.

stars Key Features

devices
Device Presets 4 presets: Android (360x800), iPhone (390x844), Tablet S (800x1280), iPad Pro (834x1194)
screen_rotation
Rotate Viewport Toggle the viewport between portrait and landscape orientation with a single click
camera_alt
Screenshot Capture Capture the current emulated viewport as a high-quality PNG screenshot image
archive
Batch Export Export screenshots for all 4 device presets at once as a single ZIP download file
search
Search Filter Quickly filter and find devices by name or screen dimensions in the preset list
keyboard
Keyboard Shortcut Open the popup instantly with Ctrl+Shift+M (or Cmd+Shift+M on Mac)

settings How It Works

1
Attach Debugger When you select a device preset, the extension attaches the Chrome Debugger API (chrome.debugger) to the active tab.
2
Override Metrics It sends Emulation.setDeviceMetricsOverride commands to resize the viewport, adjust scale, and set orientation.
3
Capture Screenshots Screenshots are captured via Page.captureScreenshot for individual captures or sequentially for batch exports.
4
Build ZIP In-Memory Batch exports compile all screenshots into a ZIP file entirely in-memory using a JavaScript ZipWriter before downloading.

favorite Privacy

verified
Your data stays yours

Goofanizer collects no data. Everything runs locally in your browser using the Chrome Debugger API. No analytics, no telemetry, no tracking, no servers. Screenshots are saved directly to your computer and are never uploaded anywhere.

download Installation Instructions

1 Click Download Extension above to get the ZIP file
2 Extract the ZIP to a folder on your computer
3 Open Chrome and go to chrome://extensions
4 Enable Developer mode (toggle at top right)
check_circle Click Load unpacked and select the extracted folder. Done!
