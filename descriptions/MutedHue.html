circle Overview

MutedHue replaces the harsh bright blue text selection on every website with a subtle, adaptive grey highlight. It is gentle on the eyes and blends beautifully with both light and dark themes -- no configuration needed, works instantly after install.

stars Key Features

contrast
Adaptive Grey Replaces harsh blue text selection with a subtle, adaptive grey on every website visited
light_mode

lock

Dynamic Detection Detects background luminance and applies the matching selection color automatically

settings How It Works

1
Inject at Document Start A content script runs at document_start to inject custom ::selection styles before the page renders, preventing the blue highlight from appearing.
2
Detect Background Luminance The script calculates the page background and switches between light and dark selection colors automatically, ensuring readability on every theme.
3
Monitor Theme Changes A MutationObserver watches the <html> element for theme attribute changes and updates the selection color in real time.
4
Patch Shadow DOM Element.prototype.attachShadow is monkeypatched to inject selection styles into Shadow DOM roots of web components.
