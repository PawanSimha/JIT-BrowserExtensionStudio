● Overview

Refreshner reloads pages at customizable intervals, monitors page content for keywords, and alerts you when matches are found. Use it for monitoring dashboards, tracking inventory, waiting for page changes, or keeping sessions alive.

★ Key Features

⏱
Preset Intervals Quick shortcuts: 10s, 30s, 1m, 5m, 30m, 1h -- tap to start
⚙
Custom & Random Set any HH:MM:SS interval or let it randomize between two values
◛
Countdown Ring Live SVG ring animation showing seconds until next refresh
✉
Keyword Alerts Scans page text and alerts via notification + audio on keyword match
⟳
Hard Refresh Bypass browser cache on each reload for fresh page content
☰
Per-Tab State Each tab runs independently with its own interval and keyword set

⚙ How It Works

Refreshner uses a background service worker with chrome.alarms to schedule tab refreshes. The content script scans page text for configured keywords on every load and reports matches back to the background worker, which dispatches chrome.notifications and plays an audio alert. The popup provides a real-time countdown UI with an SVG ring animation driven by stroke-dashoffset. All state is persisted locally via chrome.storage.local -- no data ever leaves your browser.

♥ Privacy

✓
Your data stays yours

Refreshner collects no data. All settings are stored locally via chrome.storage.local. No analytics, no telemetry, no external requests -- nothing leaves your browser. Ever.

⚙ Installation Instructions

1 Click Download Extension above to get the ZIP file
2 Extract the ZIP to a folder on your computer
3 Open Chrome and go to chrome://extensions
4 Enable Developer mode (toggle at top right)
✓ Click Load unpacked and select the extracted folder. Done!
