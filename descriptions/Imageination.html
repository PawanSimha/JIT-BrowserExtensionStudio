Download Extension

● Overview

Imageination scans any webpage for images, video, and audio, then presents them in a clean categorized popup. It detects everything from <img> tags and favicons to inline SVGs, CSS backgrounds, <video> sources and posters, and <audio> elements. Download individual files directly or batch entire categories as a single ZIP archive. A built-in color picker uses the EyeDropper API to grab colors from anywhere on screen, with persistent history saved locally. No account needed, no servers involved. Everything runs locally in your browser.

⚙ How It Works

When you open the Imageination popup on any webpage, the extension's content script scans the page DOM for all media sources: <img> elements (with natural/attribute dimension fallback), <picture> and <source> tags, inline <svg> elements (serialized to data URLs), favicon <link> tags, CSS background-image properties, <video> elements (sources + posters), and <audio> elements. Media is deduplicated by normalized URL and categorized by file extension into 21 format groups plus meta-categories (Favicon, Video, Audio). The categorized list is sent to the popup, which has two tabs: Media and Colors. The Media tab shows a left sidebar with all available categories; click any category to show its items in a 2-column grid. Download individual files via chrome.downloads.download or batch entire categories as a single ZIP archive (store method, CRC-32 validated, built entirely in-memory in the popup). The Colors tab provides a color picker using the browser's EyeDropper API, displays HEX and RGB values with a swatch preview, and saves picked colors to chrome.storage.local. Color history persists across popup sessions, supports up to 50 unique entries, and is managed by the background service worker.

♥ Privacy

✓
Your data stays yours

Imageination collects no data. Everything runs locally in your browser. No analytics, no telemetry, no tracking, no servers. The extension accesses the current tab when you click the icon, scans for media, and saves downloads directly to your computer. Color history is stored locally in chrome.storage.local and never leaves your device. Zero data ever leaves your browser.

⚙ Installation Instructions

1 Click Download Extension above to get the ZIP file
2 Extract the ZIP to a folder on your computer
3 Open Chrome and go to chrome://extensions
4 Enable Developer mode (toggle at top right)
✓ Click Load unpacked and select the extracted folder. Done!
